How to build the PADIFS solution:

- Open the PADIFS solution file and it will load the 5 projects included: (PADICommonTypes, Metadata_Server, DataServer, Client and PuppetMaster);
- Build the entire solution by pressing F6;
- Press Start and it will open the PuppetMaster window and insert the path of the script in the first textbox;
- Press LoadScript to load the file and then press Run. (Step button will work in the other release);
